Funcionamiento del programa: 

En la terminal se ejecuta:

bash menu.sh

se elige opción 1(crear entorno)
se elige opción 2 (correr proceso)

en otra terminal se ejecuta:

cp alumnos 1.txt EPNro1/entrada
cp alumnos2.txt EPNro1/entrada

volvemos a la terminal 1 
Opción 3 4 5 y 6 deberían funcionar en cualquier orden. 

Una vez que se elije la opcion 6 y se sale del programa es ideal ejecutar

bash menu.sh -d

Para borrar lo que sucedio en la ejecucion anterior y matar el proceso que se ejecuta en background.
